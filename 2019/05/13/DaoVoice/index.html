使用DaoVoice为hexo+next博客添加在线聊天功能

发表于 2019-05-13 | 分类于 hexo | 阅读次数:
字数统计: 478 | 阅读时长 ≈ 2

DaoVoice 可以提供在线联系的功能，我们可以借助于此在自己的站点上接入了此功能。

DaoVoice官网

效果预览：

右侧的聊天栏

1、注册DaoVoice

我直接使用了GitHub注册

邀请码:201adca1

公司随便写就行，规模1-5人。

2、app_id

注册成功后，进入后台控制台，“应用设置”-“安装到网站”，标题2下面右app_id，记下来。

3、配置next

以 next 主题为例（其他应该一样，不过我没试过），打开 themes/next/layout/_partials/head.swig 文件中添加如下代码，位置随意（我添加到了最后面）：

在主题配置文件 _config.yml(themes/next/_config.yml)，添加如下代码：

# Online contact 
daovoice: true
daovoice_app_id: 这里输入前面获取的app_id

4、设置聊天框有样式

进入DaoVoice后台管理页面。

“应用设置”-“聊天设置”-“定制聊天窗口样式”一栏中设置：

自定义颜色 #3890ff

位置：右下

距离右侧像素值 30.0

距离下侧像素值 65.0

点击 保存

这是我调整好的比较美观的位置。

tips：下方有“在邮件和聊天窗口下方显示“一分钟了解 DaoVoice”的字样”自己可以把勾去掉，

不过本着分享的原则，我一般都把使用的技术链接放在页面上让更多的人了解。

5、完成

hexo d -g

即可看到效果
